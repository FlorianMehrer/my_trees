# Aufgaben zu Baumstrukturen

## Binärbäume mit Integer-Werten

Im Package `bintree_int` gibt es Aufgaben zu _Binärbäumen_, deren Elemente
Zahlen als Schlüssel speichern. Dies sind die Grundlagen für den Umgang mit Bäumen.
Diese Aufgaben sollten als erstes bearbeitet werden, zumindest die in den Dateien
`element_properties_basic.go` und `element_access_advanced.go`.

## Binäre Suchbäume mit Integer-Werten

Im Package `binsearchtree_int` gibt es Aufgaben zu _binären Suchbäumen_, deren Elemente
Zahlen als Schlüssel speichern. Diese Aufgaben bauen auf den Aufgaben zu Binärbäumen auf
und die dort definierte Struktur `Element` nutzt die aus dem Package `bintree_int`.
